## Задания к лаб. 11 (лабаа11.c)
Заполнение массива с консоли – Ввод 10 элементов массива с клавиатуры
Преобразование массива – Обнуление отрицательных элементов массива
Печать массива – Форматированный вывод массива в виде таблицы с индексами, исходными и новыми значениями
Вычисление среднего арифметического – Расчёт среднего значения элементов преобразованного массива
## Задания к лаб. 14 (лаба14.c)
Заполнение массива значениями из функции – Генерация массива по формуле y = 1.8*x² - sin(10*x) на интервале [0.2; 2.2]
Печать массива – Вывод элементов массива в табличном формате
Обработка массива – Обнуление отрицательных элементов массива
Статистика обработки – Подсчёт количества обнулённых элементов

## Задания к лаб. 16 (лаб16.c)
Обработка элементов массива – Замена отрицательных элементов на ноль с использованием динамической памяти
Печать массива – Вывод исходного и преобразованного массивов
Генерация массива по функции – Создание массива значений функции 1.8*x² - sin(10*x)
Статистический анализ – Вычисление сумм и подсчёт количества положительных и отрицательных элементов, расчёт среднего значения положительных элементов

## Реализованные функции
# 1. double calculate_value(double x) (лаба14.c, лаб16.c)
Назначение: Вычисление значения функции 1.8*x² - sin(10*x)
Параметры:

x – аргумент функции
Возвращает: вычисленное значение функции

# 2. void task1() (лабаа11.c)
Назначение: Основная логика лабораторной работы №11
Функционал: Ввод массива, преобразование (обнуление отрицательных), вывод в табличном формате, вычисление среднего арифметического

# 3. void task2() (лабаа11.c)
Назначение: Генерация массива по функции и статистический анализ
Функционал: Создание массива из 100 элементов, подсчёт сумм и количества положительных/отрицательных элементов

# 4. double* full_elements(double* ptr_array, int n) (лаба14.c)
Назначение: Заполнение массива значениями функции
Параметры:

ptr_array – указатель на массив

n – размер массива
Возвращает: указатель на заполненный массив

# 5. int put_elements(double* ptr_array, int n) (лаба14.c)
Назначение: Вывод элементов массива в табличном формате
Параметры:

ptr_array – указатель на массив

n – размер массива
Возвращает: количество выведенных элементов

# 6. double* calc_elements(double* ptr_array, int n) (лаба14.c)
Назначение: Обработка массива (обнуление отрицательных элементов)
Параметры:

ptr_array – указатель на массив

n – размер массива
Возвращает: указатель на обработанный массив

# 7. void task1() (лаб16.c)
Назначение: Работа с динамическими массивами
Функционал: Создание массива через malloc, ввод данных, преобразование (замена отрицательных на ноль), вывод результатов

# 8. void task2() (лаб16.c)
Назначение: Анализ массива значений функции
Функционал: Генерация массива, подсчёт сумм положительных и отрицательных элементов, расчёт среднего значения положительных элементов
